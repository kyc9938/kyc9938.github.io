2. 내가 이해한 기획자**(혹은 생각했던 직무)**는 어떤 역할을 하는 사람인가요? 
    - 각기 다른분야의 사람들을(디자인, 개발, 마케팅) 방향성을 설정해주고 서로 협력할수있게 도움을 주는 직무 입니다

3. PM 업무를 경험해보셨나요? 해보셨다면 어떤 경험을 하셨는지를 작성해주시고,
아니라면 PM 업무에 대해 찾아본 것을 작성해주세요.
    - PM의 주요 역량은 사람들과 커뮤니케이션이 중요하며 다른 분야에 있는 사람들을 설득하고 한가지 목표를 만들어 다같이 나갈수 있게 만들어주는 중심의 역할이라고 생각합니다

4. **PM의 역할을 수행하는 데에 있어 나의 강점과 연관된 부분은 무엇이라고 생각하나요?** 
혹은 보완, 개선하고 싶은 개인 역량이 있나요 ?
    - 저는 예전부터 사람들과 일을 할 때 리더의 자리에 있었고 작업자들이 하지 못하거나 어려운 것들을 제가 배워서 했습니다
    또한 중간에 의사소통도 도맡아 하게 하고 목표설정과 과정을 이끌어 나가는 강점이 있습니다.

5. **본 코스 수료 후, 어떤 기획자(혹은 생각했던 직무)로 성장하고 싶나요?**

6. **본 코스 수료 후, 기획자(혹은 생각했던 직무)가 된 5년 후 목표로하는 점은 무엇인가요?**
